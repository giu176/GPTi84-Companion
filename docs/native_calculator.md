# Native calculator UI note

This document is superseded by the Axe-native calculator plan:

- `docs/axe_native_app.md`
- `docs/raw_axe_link_protocol.md`

The earlier pure-ASM experiment remains useful as reference code under
`programs/asm_gpti84/`, but the active calculator application direction is Axe
on TI-84 Plus monochrome. Axe gives us native screen/key/link control while
keeping the app easier to iterate than handwritten Z80.
